# TimeZone

A simple web page project for displaying and exploring time zone information.

## About

This project is a lightweight frontend app built with HTML, CSS, and JavaScript. It is designed to provide a clean interface for viewing time-related information in a timezone-aware format.

## What it uses

- HTML for the page structure
- CSS for styling
- JavaScript for interactive behavior
- dayjs for date and time handling, such as formatting and displaying time values
- micromodal for modal dialog functionality, such as opening and closing pop-up windows in the UI

## How to use

1. Open the project folder in your browser.
2. Open the file `index.html` to view the page.
3. If you want to run it locally through a development server, you can use a simple static server such as:

```bash
npx serve .
```

Then open the local URL shown in the terminal.

## Project structure

- `index.html` — main page
- `style.css` — page styling
- `package.json` — project metadata and scripts
